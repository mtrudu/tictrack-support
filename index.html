How to create a counter?

Tap the + button in the top right, choose an icon, enter a title, and select the date/time.

How to change language?

Tap the flag icon in the top left corner of "My Counters" screen.

How do notifications work?

Enable milestone notifications when creating a counter. You'll get alerts at 7 days, 30 days, 100 days, etc.

How to use "Until" mode?

Select "Until (future)" when creating a counter to count down to an event like birthdays or vacations.
